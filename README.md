# DeFi House Whitepaper (Investor Edition)
**Empowering DeFi Literacy, Financial Sovereignty, and Cross-Chain P2P Credit**

This GitBook contains the full whitepaper for the DeFi House ecosystem and the DFH token.  
It is intended for internal use and for investors participating in the DFH private sale.

A simplified **public litepaper** will be produced separately.

## Core Components
- Community-led DeFi education  
- Over-collateralized, fixed-term, fixed-rate P2P lending  
- No liquidations during the loan term  
- Cross-chain credit + liquidity expansion  
- DFH token with utility and value-accrual  
- Gradual decentralization and governance
