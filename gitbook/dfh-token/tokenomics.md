# Tokenomics

**DFH** will have:
- A **fixed total supply**  
- Allocations for *community*, *contributors*, *ecosystem development*, and *treasury*  
- **Vesting schedules** aligned with *long-term participation*  

Exact tokenomics will be published **prior to token launch**, including:
- **Distribution breakdown**  
- **Release schedules**  
- **Treasury usage guidelines**  
- *Economic modeling assumptions*
