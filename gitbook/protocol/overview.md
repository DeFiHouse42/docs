# Protocol Overview

The DeFi House protocol is a fixed-term, fixed-rate, over-collateralized P2P lending system designed to provide predictable credit.

Initial deployment will take place on BNB Chain as an MVP, followed by iterative improvements and eventual cross-chain expansion.

Core characteristics:
- Fixed-term loans  
- Fixed-rate interest  
- No liquidations during the loan duration  
- Lender-defined parameters  
- Clear settlement rules at maturity
